# Pilot App

Demo aplikacji z licznikiem rozwijane przez ai-factory.

## Funkcje

- Licznik z persystencją
- Tryb ciemny
- Logo

## W trakcie

- Statystyki sesji
- PWA

## Uruchomienie

```bash
npm ci
npm run dev
npm run build
npm test
```
